개요

daero 는 GTFS 기반 국내 대중교통 door-to-door 경로탐색 API 입니다. 인증(API 키) 불필요, 응답은 application/json; UTF-8. Base URL: http://localhost:8090

직접 호출해보기

응답이 여기에 표시됩니다.

엔드포인트

GET /api/plan/coords

좌표 door-to-door 경로. 추천 + 대안 경로 반환.

파라미터타입설명
fromLat, fromLondouble출발 좌표
toLat, toLondouble도착 좌표
timestring출발 시각 HH:MM (기본 08:00)
alternativesint대안 경로 수 0~10 (기본 1)

GET /api/plan

정류장 ID 간 경로. 파라미터: fromStop, toStop, time, alternatives.

GET /api/plan/search?q=이름&limit=20

이름으로 정류장 검색(정류장 ID·좌표 반환).

GET /api/plan/near?lat=&lon=&radius=800

좌표 근처 정류장(모드 접두어 포함).

GET /api/gtfs/stats · /api/gtfs/timetable

적재된 데이터·빌드 상태.
